What is this?

This app is one form integration between an enterprise authentication source (e.g. ADFS) and Amazon's AWS. Usually, such integration is associated with AWS console login, this app, however, gives the user an STS token, so she can make AWS calls via the command line or APIs.

Who needs this?

An organization which prefers to provision credentials ONLY in their own Active Directory, instead of having users provisioned inside AWS.

Why use it?

Because you already have a working ADFS-AWS federation and want to extend the sign-on benefits to CLI or API calls.
